Stateless Component

07 June 2017

Caveats

  1. better use functions as argument for setState function

// Correct
this.setState((prevState, props) => ({
  counter: prevState.counter + props.increment
}));

  1. reduce render times by doing magic in `shouldComponentUpdate `. try use [PureComponent](https://facebook.github.io/react/docs/react-api.html#react.purecomponent) and immutable props(especially for complex store) to avoid useless rendering.
